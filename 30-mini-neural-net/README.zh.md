# 30 — mini-neural-net

> 用纯 Python 从零实现一个前馈神经网络 —— 逻辑代码不到 200 行，无任何外部依赖。

## 背景

神经网络看起来复杂，但核心思想只有三条：

1. 把数据依次经过一系列矩阵乘法和非线性函数 —— **前向传播**。
2. 用**链式法则**反向追溯每个权重对误差的贡献 —— **反向传播**。
3. 沿着减小误差的方向微调每个权重 —— **梯度下降**。

这个项目用 Python 原生列表把三件事全部实现出来，不依赖 numpy，不依赖自动求导框架。每一条偏导数都显式写出来，让你能看清链式法则是如何穿透整个网络的。

## 架构

### 1. Layer（全连接层）

每层存储：

- `W` — 权重矩阵 `[out_size × in_size]`
- `b` — 偏置向量 `[out_size]`

**前向传播**：`z = W·x + b`，然后 `a = act(z)`。

层会缓存输入 `x` 和预激活值 `z`，因为反向传播时需要复用它们。

**反向传播**：给定上层传来的 `∂L/∂a`，链式法则推导出：

```
δ      = ∂L/∂a ⊙ act′(z)    # 与激活函数导数逐元素相乘
∂L/∂W  = δ · xᵀ              # 外积
∂L/∂b  = δ
∂L/∂x  = Wᵀ · δ              # 向前一层传递的梯度
```

### 2. Network（网络）

`Network` 就是一个 `Layer` 列表。`forward()` 从左到右依次传递数据；`backward()` 从右到左依次传递梯度，每层返回 `∂L/∂x` 给前一层。

### 3. 损失函数：MSE

均方误差及其对预测值的梯度：

```
L       = (1/n) Σ (pred - target)²
∂L/∂pred = (2/n)(pred - target)
```

这个梯度是整个反向传播的起点。

### 4. 优化器：SGD

朴素随机梯度下降，每处理一个样本就更新一次参数：

```
W ← W − lr · ∂L/∂W
b ← b − lr · ∂L/∂b
```

## 关键实现分析

### 链式法则即代码

反向传播的本质是在每层机械地应用链式法则。在 `Layer.backward()` 中：

```python
delta = [da * self.act_prime(zi) for da, zi in zip(grad_a, self._z)]
```

`grad_a` 是上层告诉当前层的信息："损失对我的每个输出的变化敏感度是多少。"再乘以 `act′(z)` 就把这个梯度从激活空间转换回预激活空间（`z`）。之后的 `dW`、`db`、`grad_x` 都直接由此推出。

### 为什么用 XOR 验证？

XOR 不是线性可分的，单层感知机无法解决。增加一个带非线性激活函数的隐藏层，网络就有了足够的表达能力。这使得 XOR 成为验证多层网络能学到单层感知机学不到的东西的最小反例。

### 激活函数

实现了两种激活函数：

| 名称    | 公式              | 导数                    |
|:--------|:------------------|:------------------------|
| sigmoid | `1 / (1 + e^-x)`  | `s(x) · (1 − s(x))`    |
| relu    | `max(0, x)`       | `1（x > 0）否则 0`      |

演示中使用 sigmoid，因为它的输出范围是 `(0, 1)`，与 XOR 的二元目标值吻合。

## 关键收获

- 神经网络本质上是一组简单函数的复合；反向传播就是对这个复合函数应用链式法则。
- 每层只需要在前向传播时保存两样东西：输入 `x` 和预激活值 `z`，其余的都可以推导出来。
- 反向传播将损失的梯度信号从输出层传回每一层，每层变换信号后继续向前传递。
- SGD 是最简单的优化器，用来解决 XOR 已经绰绰有余。更先进的优化器（Adam、RMSProp）遵循相同的更新规则，但引入了自适应学习率。
- 这份实现刻意保持慢速 —— 目标是可读性，而非性能。生产环境的网络会用 numpy 或张量库对批次维度做向量化运算。
