# mini-search

> 一个极简 Python 搜索引擎，包含倒排索引、TF-IDF 权重、BM25 排名和查询打分。

[English](README.md)

---

## 功能

- 从小型文档集合构建倒排索引
- 计算 TF-IDF 文档/查询权重，并用余弦相似度排序
- 使用 BM25 做更常见的词法检索打分
- 无第三方依赖，直接对自由文本查询返回排序结果

---

## 文件

- `src/mini_search.py`：不足 200 行的核心搜索实现
- `demo.py`：索引几篇短文并执行示例查询

---

## 运行方式

```bash
python projects/20-mini-search/demo.py
```

---

## 设计说明

这个实现刻意保持紧凑：

- `tokenize()` 负责小写化和分词
- `index[term][doc_id] = tf` 保存倒排索引
- TF-IDF 使用对数缩放词频和余弦归一化
- BM25 通过 `avgdl`、`k1`、`b` 做文档长度归一化

这样同一个索引层上就能对比两种经典 IR 排序方法，适合快速理解搜索系统的核心结构。

---

## 说明

- 这是教学用途实现，不是生产级搜索引擎。
- 没有 stemming、停用词过滤、短语检索或增量索引。
- 它更适合用来理解词法检索系统的基本组成。
