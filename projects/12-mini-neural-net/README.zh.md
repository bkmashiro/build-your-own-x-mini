# mini-neural-net

> 纯 Python 实现的极小神经网络：单隐藏层、反向传播、SGD，以及 XOR 演示。

[English](README.md)

---

## 背景

一个“足够像样”的最小神经网络，通常只需要：

- 一个全连接隐藏层
- 一个非线性激活函数
- 一个标量输出节点
- 基于梯度的参数更新

XOR 是经典玩具问题，因为单层线性分类器无法把它分开。只要加入一层隐藏层，再用反向传播训练，网络就能学到弯曲的决策边界，把四个点区分开。

---

## 架构

```text
[x1, x2]
   │
   ▼
全连接隐藏层
   │
   ▼
sigmoid 激活
   │
   ▼
输出节点
   │
   ▼
(0, 1) 概率
```

训练流程就是标准的随机梯度下降：

1. 前向传播，得到隐藏层激活值和输出概率
2. 计算预测值与目标值之间的误差
3. 把误差反向传播回隐藏层
4. 沿着梯度反方向微调权重和偏置

---

## 关键实现

### 前向传播

每个隐藏神经元先做加权求和并加上偏置，再过一层 `sigmoid`。输出节点对隐藏层结果重复同样的过程。

### 反向传播

输出层的 delta 可以写成 `prediction - target`。隐藏层的 delta 则由输出层误差、对应输出权重，以及 sigmoid 导数 `h * (1 - h)` 共同决定。

### SGD

每处理一个样本就立刻更新一次参数。这样代码很短，也最容易看清训练循环发生了什么。

## 运行方式

```bash
python projects/12-mini-neural-net/demo.py
python -m pytest projects/12-mini-neural-net/test -q
```

默认 demo 会在 XOR 数据集上训练，并打印每组输入的最终分数。

## 这里省略了什么

- NumPy 这类矩阵库
- 多隐藏层网络
- mini-batch 与 momentum
- softmax / 多分类输出
- 模型序列化

这些对更大的网络很重要，但理解前向传播、反向传播和 SGD 如何拼起来，并不需要它们。
