# mini-os

> 一个极简 Python 操作系统模拟器，覆盖 bootloader 流程、中断分发、页表地址转换和时间片轮转调度。

[English](README.md)

---

## 功能

- 像玩具 bootloader 一样启动“小内核”并加载进程镜像
- 通过最小化的中断描述表注册时钟中断和系统调用中断
- 为每个进程建立虚拟页到物理页框的映射，并通过页表完成地址翻译
- 使用简单的 round-robin 就绪队列和时间片中断做进程调度

---

## 文件

- `src/mini_os.py`：156 行核心模拟器
- `demo.py`：启动系统、运行两个进程并打印上下文切换过程

python projects/21-mini-os/demo.py

## 设计说明

这个实现刻意保持紧凑：

- `boot()` 扮演 bootloader：初始化 IDT、加载进程镜像，并把控制权交给调度器
- `interrupt(vector, ...)` 分发到时钟中断（`32`）和系统调用中断（`128`）
- `page_table[vpn] = frame` 模拟分页，`translate()` 把虚拟地址转换成物理地址
- `step()` 执行一条指令，然后触发时钟中断、处理系统调用，或结束进程

这已经足够展示启动、内存转换、中断处理和进程切换如何串在一起，而且控制流完全可见。

---

## 说明

- 这是教学模型，不是真实内核。
- 没有特权级隔离、真实设备 I/O、硬件抢占或页替换。
- 指令用字符串表示，这样更容易直接观察调度和中断流程。
