# mini-neural-net

> 从零实现前向传播、反向传播和 SGD —— 不依赖 numpy 和任何框架，约 190 行纯 Python。

**[English](./README.md)**

## 背景

神经网络在 2010 年代的崛起，源于三个看似简单却能组合爆发的核心思想：

1. **万能近似定理** —— 至少含一个隐藏层的网络，可以近似任意连续函数。
2. **反向传播** —— 对计算图递归应用微积分链式法则，使梯度计算变得高效可行。
3. **SGD** —— 对打乱顺序的 mini-batch 做梯度下降，既能逃离局部极小值，又有良好泛化性。

本实现仅用 Python 标准库中的 `math` 和 `random`，从零搭建以上全部逻辑。没有 numpy，所有矩阵运算都是列表推导式。目标是彻底看清训练过程中发生的每一步——没有黑箱。

---

## 架构

```
输入层  →  [全连接 + 激活函数] × N  →  输出层
                    ↕ 前向传播
           损失 (MSE) = Σ (pred - target)² / n
                    ↕ 后向传播
           ∂L/∂W 链式法则 → SGD 更新
```

## 关键实现细节

### 前向传播

每层计算：

```
z = x · W + b          # 线性变换 (1×in) · (in×out) → (1×out)
a = σ(z)               # 逐元素激活
```

一层的输出直接成为下一层的输入。

### 反向传播（Backpropagation）

从 MSE 的梯度出发：

```
δ_output = 2(pred - target) / n         # ∂MSE/∂pred
```

对每一层（从后往前）：

```
δ_local = δ_upstream ⊙ σ'(z)           # 与激活导数做 Hadamard 积
∂L/∂W   = xᵀ · δ_local                 # 权重梯度
∂L/∂b   = δ_local                       # 偏置梯度
δ_prev  = δ_local · Wᵀ                  # 流向上一层的梯度
```

然后 SGD 更新：

```
W ← W - lr · ∂L/∂W
b ← b - lr · ∂L/∂b
```

### 权重初始化

使用类 Xavier 初始化：`std = sqrt(2 / (fan_in + fan_out))`。这能防止早期 epoch 中激活值饱和——对 sigmoid 网络至关重要。

### 为什么用 MSE，不用交叉熵？

MSE 实现更简单，对回归任务和二分类任务已经足够。多分类任务需要 softmax + 交叉熵，但那会掩盖我们想展示的反向传播逻辑。

## 如何运行

```bash
# 无需安装任何依赖
python3 demo.py
```

演示依次运行三个经典任务：

1. **XOR** —— 2 输入、1 个隐藏层（4 个神经元），5000 个 epoch。网络学会了单层感知机无法解决的非线性 XOR 函数。
2. **正弦回归** —— 用 40 个训练点拟合 `y = sin(x)` 曲线，含 2 个隐藏层。
3. **圆形分类** —— 200 个训练点，判断二维点是否在圆内，准确率可达 99%+。

XOR 部分预期输出：

```
0 XOR 1 = 0.9846  (→1)  ✓
0 XOR 0 = 0.0087  (→0)  ✓
1 XOR 1 = 0.0239  (→0)  ✓
1 XOR 0 = 0.9832  (→1)  ✓
```

## 核心收获

- **反向传播就是链式法则** —— 逐层反向写出来，每层不过 4 行。
- **矩阵形状决定一切** —— 始终追踪 `(1×in)`、`(in×out)`、`(1×out)` 的变化，能消灭绝大多数 bug。
- **激活函数的导数需要提前缓存** —— `σ'(z)` 依赖*激活前*的值 `z`，因此前向传播时必须保存它。
- **Xavier 初始化非常关键** —— 权重初始尺度不对，梯度在第一个 epoch 就会消失或爆炸。
- **框架无魔法** —— PyTorch/JAX 做的是完全一样的事，只是加了 GPU 内核、自动微分和批次处理。数学本质相同。
